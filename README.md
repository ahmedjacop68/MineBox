# MineBox

This is a clone of minecraft Beta 1.7.3 made in SDL and OpenGL, Unlike other clones out there, This is made to be as close as possible to the original game, And be cross-platform.
* Inspired by CavEX and hatred for minecraft bedrock edition

## How to compile
Currently MineBox is very early on in development, There is only a fly camera and a test chunk, No lighting, No nothing, But if you want to try it out for some reason, This is for you:
* 1: Have MineGW installed on your PC
* 2: Clone this repostory
* 3: Go to the build folder and run:
```
make
```
